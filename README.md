# Amber Darker

Tema escuro para VS Code com paleta **âmbar/quente** sobre fundo near-black. Acompanha duas variantes que compartilham a mesma sintaxe e diferem apenas no fundo.

## Variantes

| Tema | Fundo | Indicado para |
|---|---|---|
| **Amber** | marrom-escuro quente (`#1e1812`) | quem prefere um dark mais suave |
| **Amber Darker** | near-black (`#05050b`) | quem gosta de contraste mínimo de fundo, com sintaxe levemente atenuada |

Selecione em **Preferences: Color Theme** → `Amber` ou `Amber Darker`.

## Paleta

- **Âmbar** — palavras-chave, operadores e atributos
- **Dourado** — classes, tipos e propriedades CSS
- **Areia** — strings
- **Amarelo** — definições de função
- **Branco** — chamadas de função (reduz o ruído visual em quem chama muito mais do que declara)
- **Rosa, laranja e coral** — tags, números e acentos secundários

A variante *Darker* usa tons de sintaxe um pouco mais suaves, para evitar o efeito "neon" sobre o fundo quase preto.

## Recursos destacados

- Distinção entre **definição** e **chamada** de função
- Comentários em itálico discreto
- Suporte a destaque para Markdown, diff e Git gutter
